Upload multiple files at once. Bower processes each one with the right model and turns them into searchable, editable notes.

## What you can import

Bower handles mixed batches — you don't need to separate files by type before uploading.

| File type | What Bower does |
| --- | --- |
| Images (.jpg, .png, .heic) | Text extraction — reads text, tables, and values from the image |
| Voice recordings (.m4a, .mp3, .wav) | Transcription with scientific vocabulary recognition |
| PDFs | Content extraction; auto-generates flow charts from embedded images |
| Word documents (.docx) | Converted to editable notes; formatting including tables and coloured text is preserved |

## How to run a batch import

<Steps>
  <Step title="Open the import dialog">
    In the desktop app, click **Import** in the top navigation or drag files directly into the workspace.
  </Step>
  <Step title="Select your files">
    Select all files — mixed types are fine.
  </Step>
  <Step title="Bower processes them">
    Bower queues each file and processes them in parallel. Each file becomes a separate note with the source file attached.
  </Step>
</Steps>

<Note>
There's no upper limit enforced during Beta, but very large batches (50+ files) may take a few minutes to fully process.
</Note>

## After import

- Each note is immediately searchable.
- Extracted tables are editable inline.
- The original file is always attached to the note — nothing is discarded.
- Bird can summarise, compare, or cross-reference the imported notes as soon as processing completes.

## A note on quality

Text extraction accuracy on printed text is approximately 98%. Handwritten content and instrument displays are lower — Bird will flag low-confidence extractions for your review rather than presenting them silently as verified data. You always have the final say.
