**Shipped 2026-04-19 · Brand**

A round of polish across the interface. Same product, more confident
presentation.

## What changed

- **Theme primitives** — softer input surfaces, refined borders, more
  generous whitespace on cards and lists.
- **Segmented controls** — replace tab strips and inline radios where they
  fit better. Used for theme toggle, view modes, and several settings.
- **Pill-style tabs** — the Settings and Profile pages move from
  underline-tabs to grouped pill tabs, making active state more legible.
- **Card previews** — artifact cards now follow a clean fallback chain:
  thumbnail if present, otherwise text excerpt, otherwise a quiet "No
  preview" state. No more empty boxes.
- **Per-user avatar colours** — every workspace member gets a stable colour
  derived from their identity, used consistently across the avatar,
  sidebar, search results, and tables.
- **Data table refresh** — sortable headers, calmer row dividers, and
  consistent action-icon placement.

## Why this matters

Lab software is interface-heavy by necessity. A cleaner interface lowers
the cognitive overhead between you and your work. Nothing here changes
what Bower does — it changes how it feels while you're using it.
