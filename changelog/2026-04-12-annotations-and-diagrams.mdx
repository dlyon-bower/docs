Lab work is visual. A gel image, a microscopy slide, a system diagram — these
deserve more than a caption. Bower now ships a full drawing editor inline.

## Annotate any image

Click any image in a note (or open an image attachment), tap the **draw icon**,
and the full-screen annotation editor opens with your image as the background.

- Circle the band you care about.
- Add labels with arrows.
- Highlight regions of interest.
- Sketch over a microscopy image to call out what's anomalous.

Annotations are stored as a separate layer, so the original image is never
modified — and you can re-open the editor any time to refine your drawings.

## Diagrams from scratch

Type `/diagram` in the note editor (or click the diagram icon in the
toolbar) to open the same editor with a blank canvas. Useful for:

- Workflow / decision diagrams
- Apparatus sketches
- Reaction schemes
- Quick whiteboard-style explainers

The diagram saves as an inline image in your note, and you can re-open it
to edit later — your shapes, arrows, and text are all preserved.

## What you get

The editor is powered by [Excalidraw](https://excalidraw.com/), so you get the
full set of tools: rectangle, ellipse, diamond, arrow, line, freehand draw,
text, image, eraser. Keyboard shortcuts (V, R, A, T, etc.) make it fast once
you're used to it.

## Try it

- **Annotate**: click any inline image → draw icon, or open an image
  attachment → **Annotate**.
- **Diagram**: in the note editor, type `/diagram` or click the diagram icon
  in the toolbar.
