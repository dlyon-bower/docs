**Shipped 2026-03-22 · Workspace**

We've published a Help Centre at [docs.bowerlabs.ai](https://docs.bowerlabs.ai).

It covers what Bower does, how to do it, and what's true about how your
data is handled — written to be read once and then not needed again,
because Bower should feel obvious. When something isn't, this is where to
start.

## What's in it

- **Capture** — voice transcription, photo OCR, batch import, the note
  editor, image annotations, version history, keyboard shortcuts.
- **AI & Search** — Bird, Live Mode, hybrid semantic search, personal
  preferences, workspace instructions, inferred links.
- **Organisation** — collections, protocols, tags, linked entities,
  sharing, export.
- **Workspaces & Account** — creation, invites, permissions, account
  settings, dark mode, feedback.
- **Trust Centre** — encryption, sub-processors, audit logs, HIPAA, GDPR,
  privacy and terms.

## In-app links

The artifact lists and feature surfaces inside Bower now link to the
relevant Help Centre page from a "Learn more" affordance — so the docs are
one click away from the place you'd need them.
