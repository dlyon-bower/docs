**Shipped 2026-02-26 · Search**

Press **Cmd+K** (or **Ctrl+K** on Windows) anywhere in Bower and the command
palette opens. Search every note, attachment, collection, protocol, and tag
in your workspace from one place — and run quick actions without digging
through menus.

## What's in there

- **Global search** across all artifact types, with hybrid keyword + semantic
  ranking so you find things by meaning, not just exact words.
- **Move and copy** — pick any artifact, search for a destination collection,
  and place it without leaving the keyboard.
- **Jump to** — direct navigation to any artifact you've recently touched.
- **Quick actions** — create a note, start a voice capture, open settings.

## Try it

Press **Cmd+K** from any page. Type the first few letters of what you're
looking for and the matches appear as you type.
