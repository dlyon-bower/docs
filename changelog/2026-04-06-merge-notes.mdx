**Shipped 2026-04-06 · Workspace**

You captured the same observation twice. Or three voice notes that should
really be one. Or two notes that grew into the same experiment from
different angles. Merge handles those.

## What's new

- **Pick a target, pick a source** — choose which note absorbs the other.
  The merged result keeps the target's title and metadata; the source's
  content, attachments, tags, and links flow in.
- **Content concatenation** — the source's body is appended to the target
  with a clear divider so you can see what came from where, then edit
  freely.
- **Attachments and inline images** preserved.
- **Tags and links de-duplicated** — if both notes had the same tag or
  linked the same protocol, you get one of each, not two.
- **Source moves to trash** — recoverable for 30 days if you change your
  mind, then auto-purged.

## Try it

Open the source note → **⋮ menu → Merge into…** → pick the target. Bird's
also been taught about merging, so you can ask "merge this with my note
from Tuesday" in chat.
