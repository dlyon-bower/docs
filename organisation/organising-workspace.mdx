Bower gives you a few ways to move, copy, and tidy up your workspace. None of them require you to open a note to act on it.

## The page tree

The left-hand panel is your page tree — a hierarchical view of all your notes and collections. It's the main way to navigate and organise your workspace.

### Drag and drop

- **Move a note** — drag it from its current position to a new location in the tree. Drop it onto a collection to nest it inside.
- **Reorder notes** — drag up or down within a collection to change the order.
- **Move a collection** — drag an entire collection (and everything inside it) to a new position.

Drag-and-drop works for both notes and collections.

## Multi-select bulk actions

For moving, copying, or deleting multiple items at once:

<Steps>
  <Step title="Select items">
    Cmd+click (Mac) or Ctrl+click (Windows) to select multiple notes or collections.
  </Step>
  <Step title="Right-click any selected item">
    This opens the bulk action menu.
  </Step>
  <Step title="Choose an action">
    - **Move to** — move all selected items to a collection
    - **Copy to** — copy all selected items to a collection (originals stay in place)
    - **Delete** — remove all selected items
    - **Add to collection** — associate all selected items with a collection without moving them
  </Step>
</Steps>

Bulk actions apply to everything in your selection. If you've selected a collection, all notes within it are included.

## The **&#8942;** menu

Every note and collection in the page tree has a **&#8942;** menu (the three-dot icon that appears on hover). This gives you per-item actions without needing to open the item first.

| Action | What it does |
| --- | --- |
| Rename | Edit the note or collection title inline |
| Move to | Move to a different collection |
| Copy to | Duplicate to a collection |
| Add to collection | Associate with a collection |
| Delete | Remove permanently (with confirmation) |
| Share | Manage access for workspace members |

## A note on deletion

Deleted notes are permanently removed and cannot be recovered. Original attachments are deleted alongside their notes. If you need to preserve data, [export it](/organisation/export) before deleting.
