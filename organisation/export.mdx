---
title: "Exporting notes"
description: "Export your notes as PDF, Markdown, HTML, or plain text."
---

You can export any note from Bower in four formats. The original content and formatting are preserved as closely as each format allows.

## Export formats

| Format | Best for |
| --- | --- |
| **PDF** | Sharing with collaborators, printing, archiving |
| **Markdown** (.md) | Version control, plain text workflows, other tools |
| **HTML** (.html) | Web publishing, embedding in reports |
| **Plain text** (.txt) | Maximum compatibility, clipboard use |

## How to export

1. Open the note you want to export.
2. Open the note's action menu.
3. Select the export format you want.
4. The file downloads with the note's title as the filename.

For **PDF**, Bower opens your browser's print dialog with the note pre-formatted for A4 pages. Use "Save as PDF" from the print dialog.

For **Markdown, HTML, and plain text**, the file downloads directly.

## What's included

- Full note content with formatting (headings, lists, tables, bold/italic)
- Images (embedded in HTML and PDF; referenced in Markdown)
- Task list states (checked/unchecked)

Attachments are not included in the export — only the note's text content. If you need the original source files, download them separately from the note's attachment panel.
